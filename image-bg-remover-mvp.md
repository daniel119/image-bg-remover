# Image Background Remover — MVP 需求文档

**版本：** v0.1  
**日期：** 2026-03-18  
**作者：** 刘孔灿  

---

## 1. 产品概述

一个轻量级在线图片背景去除工具，用户上传图片后自动去除背景，支持下载透明底 PNG。

**核心价值：** 免安装、免注册、秒出结果。

---

## 2. 技术栈

| 层级 | 技术 |
|------|------|
| 前端托管 | Cloudflare Pages |
| 后端逻辑 | Cloudflare Workers |
| AI 处理 | remove.bg API |
| 图片存储 | 无（纯内存流转） |

---

## 3. 用户故事

| # | 作为... | 我想要... | 以便... |
|---|---------|-----------|---------|
| 1 | 访客 | 上传一张图片 | 去除背景 |
| 2 | 访客 | 预览去背景后的效果 | 确认结果满意 |
| 3 | 访客 | 一键下载透明底 PNG | 直接使用 |

---

## 4. 功能范围

### ✅ MVP 包含

- 图片上传（点击选择 + 拖拽）
- 调用 remove.bg API 处理
- 结果预览（原图 / 处理后对比）
- 下载按钮（PNG 透明底）
- 错误提示（上传失败 / API 错误）
- 加载状态动画

### ❌ MVP 不包含

- 用户注册 / 登录
- 历史记录
- 批量处理
- 图片编辑（裁剪、替换背景等）
- 付费墙

---

## 6. 接口设计

### Worker API

**POST** `/api/remove-bg`

- Request: `multipart/form-data`，字段名 `image_file`
- Response 成功: `image/png` 二进制流
- Response 失败: `application/json` `{ "error": "..." }`

**限制：**
- 文件类型：JPG / PNG / WebP
- 文件大小：≤ 10MB（前端校验）

## 7. 非功能需求

| 项目 | 目标 |
|------|------|
| 处理时长 | < 5 秒（取决于 remove.bg） |
| 移动端适配 | 响应式，支持手机上传 |
| 安全 | API Key 存于 Worker 环境变量，不暴露前端 |
| 成本 | Cloudflare 免费套餐 + remove.bg 按需付费 |
